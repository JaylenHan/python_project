# 프로젝트 개요 (Project Brief)

## 한 줄 요약

대학 캠퍼스를 배경으로 한 2D 턴제 RPG. 강남대학교 파이썬응용 기말고사 프로젝트로 개발되어 **학부 우수작품으로 선정**되었다.

## 배경 / 기획 의도

- **RPG 게임의 매력적 요소 결합** — 포켓몬·메이플스토리 같은 인기 RPG의 모험·성장 경험을 도입
- **개성 있는 캐릭터와 이야기** — 캐릭터마다 고유 능력과 스토리를 부여해 다양한 플레이 방식 제공
- **현실과의 공감대 형성** — 실제 대학교 생활을 모티브로 현실적이고 친근한 환경을 구성, 학우 간 공감과 몰입감 강화

## 범위

### 구현됨 (In Scope)

- 4종 플레이어 캐릭터 선택 및 전용 아트
- 격자 기반 캠퍼스 맵 탐험(7개 지역) + 상호작용 칸 + 랜덤 상자
- 턴제 전투(기본 공격 / 게이지 스킬 / 아이템 / 도망)
- 점층형 보스전 3종 + 전후 대화 분기(승리/패배)
- 미니게임 3종(과녁 맞히기 / 팩맨 / 학점 게임)
- 상점 및 아이템 경제(코인, HP/MP/에너지 물약, 능력 강화)
- 지역별 BGM · 보스 테마 · 공격 효과음
- 6컷 인트로 만화 및 대화 시스템

### 미구현 (Out of Scope / 향후)

- 캐릭터 커스터마이징
- 미니게임·아이템 추가 확장
- 멀티플레이어
- 스토리·퀘스트 확장
